SmartPayroll Web Application

The final product of the SmartPayroll ETL project is a fully functional payroll management web app developed using Flask and SQLAlchemy. It provides a user-friendly interface to interact with the processed payroll data stored in SQL Server.

  • 🔍 Employee Search & Filter: Easily search employees by name or ID, and filter by department.
  • 🏢 Department View: View all employees under a department with summary stats.
  • 💰 Salary Breakdown: Net salary is calculated using allowances, deductions, and basic salary logic.
  • 📥 CSV & PDF Downloads: Export payroll data for reports or record-keeping.
  • 🌙 Dark Mode Toggle: Switch between light and dark themes for better accessibility.

Conclusion

SmartPayroll demonstrates how real-world payroll data can be transformed via an ETL pipeline, analyzed with Python, and delivered through a full-stack web application.
