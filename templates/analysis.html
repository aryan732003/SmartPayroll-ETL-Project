Data Analysis in Jupyter Notebook

After successfully executing the ETL pipeline, the cleansed payroll data was stored in SQL Server. Using Python and Pandas within a Jupyter Notebook, we performed comprehensive data analysis to extract actionable insights and trends from the employee dataset.

  • Data Cleaning: Handled missing values, ensured correct data types, and removed duplicates or outliers.
  • Salary Insights: Identified high and low salary ranges, department-wise distributions, and anomalies.
  • Allowance Distribution: Explored how different departments allocate allowances and benefits.
  • Visualizations: Generated visual reports like bar graphs, pie charts, and histograms using matplotlib and seaborn.
