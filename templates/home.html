Welcome to SmartPayroll ETL

SmartPayroll ETL is a comprehensive end-to-end payroll management system developed using a modern data engineering and web development stack. This project highlights how real-world payroll data can be extracted, transformed, analyzed, and displayed seamlessly through an automated ETL pipeline and interactive web application.

At its core, the project leverages a structured ETL (Extract, Transform, Load) process: raw data in CSV format is ingested via Python scripts, cleaned and transformed, and then loaded into a relational database—Microsoft SQL Server. Python's pandas, pyodbc, and SQL stored procedures are used to ensure the data is accurate, relational, and optimized for queries.

The processed data is then analyzed in Jupyter Notebook using Python’s data science libraries such as pandas, matplotlib, and seaborn. These notebooks uncover meaningful insights—like salary distributions, department-wise expenses, and leave patterns—that help in making strategic decisions.

The final step is visualization and user interaction. The project uses the Flask web framework to build a lightweight and responsive web interface. The Flask backend connects to the SQL Server using SQLAlchemy, and dynamically fetches data to power pages like:

  • Employee search & filter
  • Department-based breakdowns
  • Payroll details and salary breakdown
  • CSV and PDF downloads for payroll reports
  • Dark mode toggle on detail views

This end-to-end solution demonstrates how data engineering (ETL pipelines), data analysis (Python/Jupyter), and full-stack development (Flask + SQL Server) can work together to solve real business problems with technology. Every step from raw data to final interface has been built from scratch for full transparency and control.
